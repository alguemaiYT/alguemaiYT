### 💫 About Me
I am an 18-year-old developer passionate about bridging the gap between physical hardware and advanced software. I specialize in building end-to-end systems: from tuning PID controllers and writing C++ firmware for ESP32, to deploying YOLO ONNX models and LLM agents on ARM Linux boards.

* 🔭 **Currently working on:** Project G.E.R.A (C++ firmware & control for autonomous sumo robots) and NoNail (MCP Agent with multi-LLM integration).
* 🌱 **Focus:** Transforming bench prototypes into highly responsive, functional systems.
* 📫 **Reach me at:** cooprevisaude@gmail.com
